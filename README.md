## Quick start

You only need [NodeJS](https://nodejs.org/en/).

```bash
git clone git@github.com:jackfruitgames/jackfruit.games.git
cd jackfruit.games
npm i
npm run start
```

The page is hosted on GitHub Pages.

```bash
# build and deploy to gh-pages branch
npm run deploy
```
